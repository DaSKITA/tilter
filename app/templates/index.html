Übersicht

Diese Annotationsplattform ist Teil eines neuen Ökosystems zur Verarbeitung von datenschutzrechtlichen Transparenzinformationen. Sie unterstützt den Prozess der Überführung von textuellen Datenschutzerklärungen in das strukturierte, maschinenlesbare Format TILT, welches in einem Dokumentenspeicher abgelegt werden kann. Später können die so „gewonnenen“, also zur Weiterverarbeitung aufbereiteten, Informationen über verschiedene Kanäle ausgespielt werden. Beispielhaft werden im Projekt DaSKITA bereits eine Browser-Erweiterung oder ein interaktiver dialogbasierter Chatbot entwickelt.

Weitere Informationen: https://daskita.github.io
